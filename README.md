# Loot Survivor Filter

Transform any image into Loot Survivor's signature neon green aesthetic!

## Features
- 5 tier presets matching Loot Survivor equipment rarity
- Real-time image filtering
- Download filtered images
- Share directly to X (Twitter)
- 100% client-side processing

## Live Demo
Coming soon on Vercel!

## Development
Simply open `index.html` in a browser or run a local server:
```bash
python3 -m http.server 8000
```
